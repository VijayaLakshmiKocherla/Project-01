Project Name: Become a millionaire

Description: "Who wants to be a millionaire" is a game in which the contestants need to register first, and then login to start the game.

This game is a quiz competition in which the goal is to correctly answer a series of fifteen consecutive multiple-choice questions. Each question is worth a specific amount of money and the money increases as the contestant moves to next question. If at any time the contestant gives a wrong answer, the game is over and the contestant will receive whatever money he/she had won up to that point. If the contestant answers all the questions correctly, then he wins a million.

A 50-50 lifeline is available for a contestant to use if a question proves difficult. With this lifeline, Two incorrect answers are eliminated, leaving the contestant with a choice between the correct answer and one remaining incorrect answer.
